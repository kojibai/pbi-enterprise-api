# Contributing

By contributing, you agree that your contributions may be used under:
- AGPL-3.0-or-later, and
- Kojib commercial licensing terms.

All contributors must sign the CLA in CLA.md (or use a CLA bot if enabled).

## Development
- Node >= 18
- `npm install`
- `npm run build`
- `npm run demo`

## Security
If you discover a security issue, do NOT open a public issue. See SECURITY.md.
